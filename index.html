URL Inspection Tool

Follow these steps to use the URL Inspection Tool:

  1. Enter your Google Cloud Client ID and API Key.
  2. Click the "Initialize" button to set up the API client.
  3. Click the "Authorize" button to sign in with your Google account.
  4. Enter the site URL you want to inspect. This should match a property you have verified in Google Search Console.
  5. Paste the URLs you want to inspect, one per line, in the text area below.
  6. Click the "Inspect URLs" button to start the inspection process.
  7. Once the inspection is complete, you can view the results on the page or export them as a CSV file.

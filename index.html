URL Inspection Tool

Follow these steps to use the URL Inspection Tool:

  1. Enter your Google Cloud API Key. You can obtain this from the Google Cloud Console.
  2. Enter the site URL you want to inspect. This should match a property you have verified in Google Search Console.
  3. Paste the URLs you want to inspect, one per line, in the text area below.
  4. Click the "Inspect URLs" button to start the inspection process.
  5. Once the inspection is complete, you can view the results on the page or export them as a CSV file.

Note: Keep your API key confidential. Do not share this page with your API key filled in.
